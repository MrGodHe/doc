# 问题：centos服务器  /dev/mapper/centos-root   100%

1. 总体磁盘空间容量查询

   ```
   df -h 
   df -ah
   ```

2. 查看当前文件夹大小

   ```
   #不列出子项
   du -sh 
   #列出子项
   du -sh *
   ```

3. 查看当前目录所有文件及文件中包含的子文件夹大小

   ```
   #若是顶级目录，显示很多，不利查看 --- （建议用4命令查询）
   du -ch
   ```

4. 查看当前目录所有文件和文件夹大小（不列出子目录）

   ```
   #a所有，h表示以人类能看懂的方式，max-depth表示目录的深度
   du -ah --max-depth=1
   ```
