# 检查系统版本

```
uname -i
```

# 下载JDK

  下载地址：https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

   注：需要登录Oracle账号，可使用临时账号 http://bugmenot.com/view/oracle.com

# 卸载JDK

1. 检查已经安装的JDK

   ```
   rpm -qa|grep jdk
   ```

2. 卸载已安装JDK

   ```
   yum -y remove xxxjdk
   ```
